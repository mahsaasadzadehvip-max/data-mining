# data-mining
## 📊 Course Project

As part of my Bachelor's course in Data Analytics at Luigi Vanvitelli University (with Prof. Rossana Verde), I developed a project based on the concepts learned throughout the course.

### 🔍 Project Focus:
- Applying data mining techniques to real-world scenarios
- Understanding how to extract patterns and insights from data
- Exploring customer behavior and data-driven decision-making

### 🧠 What I Practiced:
- Data preprocessing and cleaning
- Identifying patterns in datasets
- Using analytical thinking to support business strategies

This project helped me connect theoretical concepts with practical applications in data analysis.
